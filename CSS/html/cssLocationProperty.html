디스플레이 속성

p - 블록 속성
p 태그는 기본적으로 마진값을 갖고있다.

a - 인라인 속성
div - 기본적으로 블록 속성
div는 기본적으로 블록 속성을 갖고있어서 inline-block 으로 display를 바꿔줘도 width, height를 블록처럼 변화시킬수 있다. 반면, a는 인라인속성이라 width, height 값을 줘도 변하지 않는다.

Visibility 속성확인

기본속성은 visible

hidden으로 숨김

none으로 없앰

Opacity 속성

opacity로 투명도 조절 : 0에 가까울 수록 투명해짐

Position 속성 확인

정적위치
상대위치 : 원래 가진 정적위치 값에서 이동한 위치
절대위치 : 값이 고정된 Top과 right에서는 위치가 고정
브라우저의 크기에 따라서 bottom과 left는 위치가 변경
고정위치 : 브라우저 화면 상에서 고정된 위치

z-index는 기본값을 0으로 가지고있고ㄴ 값이 크면 앞으로 작으면 뒤로 간다.
각 요소들인 작성해준 순서대로 쌓인다. 가장 마지막에 작성해준 요소가 제일 위에 출력
